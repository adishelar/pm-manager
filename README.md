# PM Manager

This is a ** Management Scheduler** application with a React frontend and Spring Boot backend.

**Frontend deployed on GitHub Pages:**  
[https://adishelar.github.io/pm-manager/](https://adishelar.github.io/pm-manager/)

**Backend deployed on Render (Spring Boot + H2 DB):**  
[https://pm-manager-y8b1.onrender.com](https://pm-manager-y8b1.onrender.com)

> ⚠️ Note: The backend is hosted on a free Render plan,so it runs sometimes ,and sometimes it donsnt .

---

## Features

- Login for Supervisors
- View and manage tasks
- Dynamic routing for multiple pages
- H2 database support for easy setup
- Fully responsive frontend
